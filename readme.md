###Welcome to boxdeluxe.io

##System Dependencies
#Node & NPM, of course
```bash
$ brew install node
```
#Yeoman
[Yeoman](http://yeoman.io/) is a scaffolding tool
```bash
$ npm install -g yo
```
#Gulp
[Gulp](http://gulpjs.com/) for task automation and build
```bash
$ npm install -g gulp
```

##Project dependencies
```bash
$ npm install
```

##Running the dev environment
```bash
$ gulp
```
